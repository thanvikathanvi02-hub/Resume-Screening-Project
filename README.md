 📄 Resume Screening System using Machine Learning

 🚀 Project Overview

This project is a Machine Learning based Resume Screening System that automatically classifies resumes into job roles such as Data Scientist, Web Developer, Backend Developer, and UI Designer using Natural Language Processing (NLP) and a Naive Bayes classifier.

---

 🎯 Objective

- Automate resume shortlisting  
- Analyze resume text data  
- Predict job roles using ML  

---

 🛠️ Technologies Used

- Python  
- Google Colab  
- Pandas & Matplotlib  
- Scikit-learn  
- NLP (CountVectorizer)  
- Naive Bayes Classifier  

---

 🧠 Workflow

- Load dataset  
- Text preprocessing  
- Feature extraction (CountVectorizer)  
- Model training  
- Prediction  
- Evaluation  

---

 📊 Visualizations

- Model accuracy  
- Prediction output  
- Confidence distribution  
- Dataset balance

 💡 Features
